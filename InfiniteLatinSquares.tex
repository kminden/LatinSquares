\begin{theorem}
Assume $\CH$. The group $(\R, +)$ has a directed $T_\infty$-terrace $g: \R \To \R$. 
\end{theorem}
\begin{proof}
Consider the poset $\P$ of partial, directed, countable $T_\infty$-terraces, partially ordered so that $a \leq c$ if and only if $\dom a \subseteq \dom c$ and $c \rest \dom a = a$. Each condition $a \in \R$ should satisfy the property that in fact for each $d \in \R$, the function $b_{(d)}: \dom a \To \ran a \setminus \{0\}$ defined by $b_{(d)}(i)=a(i+d)-a(i)$ is a bijection.

Need to establish: \begin{enumerate}
	\item $\P$ is countably closed.
	\item It is dense to add a real number $r$ to the range of a condition.
\end{enumerate}
\end{proof}
